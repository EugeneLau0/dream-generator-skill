# 🌙 Dream Generator Skill

> 将词语、碎片、情绪与意象，织成一场完整的梦。

---

## 什么是 Dream Generator？

**Dream Generator** 是一个 Qoder Agent Skill，专门将用户提供的关键词、情绪、意象或风格要求，生成具有文学感的中文梦境叙事。

输出不是故事摘要，而是真实的梦：感官细节丰富、逻辑松动、情绪连贯、以余韵收尾。

---

## 快速上手

在 Qoder 中安装此 Skill 后，直接描述你的梦境素材即可：

```
月亮、地铁、旧朋友
```

```
写一个温柔但有点诡异的梦：雨夜、白猫、电话亭
```

```
用"失重、蓝色、找不到出口"写一个短梦
```

```
来一个像噩梦但不血腥的梦，关键词：电梯、镜子、空学校
```

无需额外说明，Skill 会直接进入梦境，不做分析或解读。

## 输入方式

| 输入类型 | 示例 |
|---|---|
| 意象关键词 | `月亮、地铁、旧朋友` |
| 情绪 + 意象 | `孤独感、旧家、消失的人` |
| 风格 + 意象 | `写一个赛博朋克风格的梦：霓虹、数据庙、雨` |
| 长度限制 | `200 字以内的短梦` |
| 风格限制 | `不要太恐怖` / `更有诗意` / `日记体` |
| 随机生成 | `给我一个意外之梦` |

---

## 可选风格

| 风格 | 效果 |
|---|---|
| 温柔疗愈 | 安静、宽和、细腻的温暖 |
| 诡异悬疑 | 压抑感、循环空间、错误的镜像 |
| 童话寓言 | 象征性强、物品会说话、规则与代价 |
| 赛博霓虹 | 孤独、人造光、身份迷失 |
| 旧日记忆 | 怀旧、不可靠的重返、消逝的熟悉感 |
| 荒诞黑色幽默 | 官僚噩梦、平静接受荒谬 |
| 孤独诗意 | 漂移、将至未至、缺席变得可触摸 |
| 史诗奇幻 | 宏大神话感、预兆、门槛与变形 |

---

## 默认行为

- 第一人称叙述
- 400–800 汉字（可指定长度）
- 默认中文（可指定其他语言）
- 直入场景，不使用"我梦见""醒来前"等框架语
- 结尾开放，以意象收尾，不做道德说教或解释
- **不主动展示评分或分析**，除非你明确要求

## 示例输出

**输入：** `月亮、地铁、旧朋友`

> 末班地铁的车厢里没有广告，只有一轮月亮贴在每扇窗外，像列车一直绕着同一个夜晚行驶。到第三站时，一个很久没见的旧朋友坐到我对面，手里拿着我们小时候传过的纸条，可纸条展开后是一张地铁线路图。每个站名都是我们曾经说过又忘掉的话……那轮月亮慢慢降到座位底下，照亮了一枚没有投出的硬币。

---

**输入：** `写一个 200 字以内的短梦：玻璃湖、童年、钟声`

> 童年的夏天多了一片玻璃湖，湖面透明得能看见云在下面游。远处钟声一响，小时候的我就从湖底跑过来，手里攥着一颗还没融化的糖……钟声越来越近，湖面碎成一块块窗玻璃，最后一下回音停在我的掌心。

## 安全边界

Dream Generator 允许怪异、恐惧与情绪强烈的内容，但默认保持文学性而非猎奇感：

- 暴力转化为氛围、余像与暗示
- 噩梦保持压迫感，不产出猎奇血腥
- 不对梦境内容给出心理或医学诊断
- 不生成涉及未成年人的性内容或任何自伤指引
