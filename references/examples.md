# Dream Generator Examples

Use these as shape references, not fixed templates. Keep final outputs responsive to the user's exact request.

Prefer immersive openings over explicit frames such as "我梦见". Prefer resonant endings over "醒来前/醒来时".

## Minimal Motifs

User: 月亮、地铁、旧朋友

Assistant:
末班地铁的车厢里没有广告，只有一轮月亮贴在每扇窗外，像列车一直绕着同一个夜晚行驶。到第三站时，一个很久没见的旧朋友坐到我对面，手里拿着我们小时候传过的纸条，可纸条展开后是一张地铁线路图。每个站名都是我们曾经说过又忘掉的话。她问我要不要下车，我低头发现鞋底沾着月光，像踩过一片很浅的水。车门打开，外面不是站台，而是学校操场。广播里念着我的名字，却用的是她的声音。那轮月亮慢慢降到座位底下，照亮了一枚没有投出的硬币。

## Abstract Motifs

User: 蓝色、失重、找不到出口

Assistant:
整栋楼都被漆成蓝色，连空气也像刚洗过的玻璃。我在楼梯间往下走，脚却越来越轻，最后每跨一级台阶都要在空中停很久，像有人把重力调慢了。门牌号从七楼跳到海拔，又跳到一个我不认识的年份。我推开每一扇门，里面都是同一间房，只是窗外的天空越来越低，低到贴着桌面流动。我知道出口一定在某处，因为风一直从背后吹来，可我回头只看见自己的影子漂在天花板上。那把蓝色钥匙轻得像一声叹息，刚碰到锁孔就浮了起来。

## Style Request

User: 写一个温柔但有点诡异的梦：雨夜、白猫、电话亭

Assistant:
雨夜的街道很安静，所有店铺都提前睡着了，只有一座旧电话亭亮着柔和的黄光。亭子里蜷着一只白猫，尾巴绕住听筒，像在等一个不会再打来的电话。我推门进去，雨声立刻被关在外面，世界只剩下猫的呼吸和线路里的细小电流。电话忽然响了，白猫抬头看我，眼睛里映着一间我小时候住过的房间。听筒那边没有人说话，只有杯子放到桌上的声音，熟悉得让我鼻子发酸。后来白猫跳到我肩上，爪子很暖，电话亭却开始慢慢下沉，像一盏灯沉进雨水深处。听筒里有人轻轻说：别淋湿了。

## Short Dream

User: 写一个 200 字以内的短梦：玻璃湖、童年、钟声

Assistant:
童年的夏天多了一片玻璃湖，湖面透明得能看见云在下面游。远处钟声一响，小时候的我就从湖底跑过来，手里攥着一颗还没融化的糖。她隔着湖面对我笑，问我为什么长这么高却还是找不到回家的路。钟声越来越近，湖面碎成一块块窗玻璃，最后一下回音停在我的掌心。

## Nightmare Constraint

User: 写一个像噩梦但不要血腥的梦：电梯、镜子、空学校

Assistant:
夜里的空学校一盏盏亮起走廊灯，却没有照出任何影子。尽头有部电梯，按钮上不是楼层，而是语文、数学、毕业、忘记。我按下毕业，门合上后，四面墙全变成镜子。镜子里的我没有看我，而是齐齐望向身后。电梯一直上升，广播却说正在下降，请不要回头。门终于打开，外面还是同一条走廊，只是每间教室里都坐着背对黑板的空课桌。镜子里有个声音替我回答了点名。
